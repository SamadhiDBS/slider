# Simple Image Slider in HTML, CSS & JavaScript

A clean and responsive **image slider/carousel** built using **HTML, CSS, and JavaScript**.  
Showcases multiple slides with images, titles, descriptions, and navigation arrows.  

---

## Features
- Smooth **image transitions** and sliding effect  
- **Navigation arrows** to move between slides  
- **Thumbnail navigation** for quick access to slides  
- Responsive layout suitable for **portfolio or presentation**  
- Includes **buttons** for actions like "See More" or "Download CV"  

---

## Demo
You can see the slider in action by opening `index.html` in your browser.

---

## How to Use
1. Clone or download this repository.  
2. Ensure the **images** are in the `image/` folder.  
3. Open `index.html` in your preferred web browser.  
4. Navigate through slides using the **arrows** or **thumbnails**.
